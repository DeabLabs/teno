# 📝 Teno

Teno is a Discord bot that records voice channels and saves them as transcribed text files.

## Usage

```sh-session
# Clone teno, and then inside the directory:
$ npm i -g yarn
$ yarn
$ yarn build

# Set up tokens (see .env.example)
$ cp .env.example .env
$ code .env

# Start the bot!
$ docker compose up -d
$ yarn start
```
